<h2 style="display: flex; flex-direction: row; justify-content: start;"> <img width="15px" alt="Abod1960" src="https://i.imgur.com/ZGdXKdI.png"> ¿Qué es Kingdom Hall Attendant?</h2>

**Kingdom Hall Attendant** es la herramienta digital esencial para superintendentes de circuito, ancianos y siervos ministeriales. Diseñada con una elegancia inigualable, KHA ofrece una experiencia fluida en una amplia gama de dispositivos y navegadores, desde Windows y macOS hasta Linux y los principales navegadores como Firefox, Chrome, IE, Edge y Safari. Pronto, podrás disfrutar de su funcionalidad también en dispositivos móviles con aplicaciones nativas para Android, iPhone y iPad.

Con KHA, simplificar la gestión de tu congregación nunca ha sido tan intuitivo. Registra informes de predicación, gestiona asignaciones y recibe recordatorios con facilidad, todo con un diseño pensado para optimizar cada tarea y responsabilidad.

La seguridad y privacidad son fundamentales para nosotros. Gracias a avanzadas funciones de protección de datos, puedes estar seguro de que la información de tu congregación está protegida en todo momento.

Kingdom Hall Attendant es completamente gratuito, sin costos ni donaciones obligatorias. Para quienes deseen apoyar el proyecto, ofrecemos opciones de financiamiento opcionales: **Season Pass**, que desbloquea temas y estilos exclusivos según la estación del año, y **Golden Edition ✨**, un gesto de generosidad que incluye un tema dorado único. Estas opciones son completamente opcionales; la aplicación en sí es gratuita y siempre lo será, con actualizaciones de seguridad continuas para una experiencia segura y confiable.

Experimenta la potencia de Kingdom Hall Attendant en cualquier momento, en el sistema operativo que prefieras. Creada con HTML y tecnología Python, esta solución multiplataforma redefine la gestión de congregaciones. Optimiza tu trabajo y transforma la manera en que gestionas tu congregación con KHA.

#### Instalación de 60 segundos

1. Clonar el proyecto

```bash
  git clone https://github.com/livrasand/Kingdom-Hall-Attendant
```

2. Ir al directorio del proyecto

```bash
  cd C:\Users\{user}\Kingdom-Hall-Attendant
```

3. Instalar dependencias

```bash
  pip install -r requirements.txt
```

4. Iniciar el servidor

```bash
  python app.py
```
