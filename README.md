<h2 style="display: flex; flex-direction: row; justify-content: start;">¿Qué es Kingdom Hall Attendant?</h2>

Kingdom Hall Attendant fue creado para ofrecer una herramienta gratuita, segura y sin anuncios, que funcione en cualquier dispositivo y navegador. Su objetivo es hacer que las tareas administrativas de las congregaciones sean más fáciles de manejar, permitiendo que el tiempo se dedique a la misión principal. No solo hemos diseñado una aplicación; hemos construido una herramienta pensada para ser intuitiva, accesible y que pueda ser usada por cualquier congregación sin complicaciones.

KHA es una aplicación multiplataforma que funciona en Windows, macOS, Linux, iPads y tabletas Android, con acceso desde cualquier navegador web. La facilidad de uso y la seguridad han sido nuestras prioridades desde el primer día. Cada aspecto ha sido diseñado para garantizar que puedas acceder a tus herramientas administrativas donde y cuando las necesites. 

Nuestro compromiso es que KHA será siempre gratuito, sin suscripciones ocultas ni fechas de caducidad. Además, nos tomamos muy en serio la seguridad y privacidad de los datos: están cifrados y nunca son rastreados. A través de un sistema seguro de transmisión de datos, protegemos tu información de manera eficaz.

<h2 style="display: flex; flex-direction: row; justify-content: start;">Ejecutar localmente</h2>

#### Instalación de 60 segundos

1. Clonar el proyecto

```bash
  git clone https://github.com/livrasand/Kingdom-Hall-Attendant
```

2. Ir al directorio del proyecto

```bash
  cd C:\Users\{user}\Kingdom-Hall-Attendant
```

3. Instalar dependencias

```bash
  pip install -r requirements.txt
```

4. Iniciar el servidor

```bash
  python app.py
```         

## ¿Funciona Kingdom Hall Attendant en mi idioma?

**¡Sí!** Kingdom Hall Attendant ha sido traducido a varios idiomas por numerosos voluntarios; por lo tanto, puedes configurar el idioma que deseas mostrar en la interfaz de la aplicación. Algunas de estas traducciones están realizadas por IA, por lo que es posible que no sean de la mejor calidad. Para ayudar, corrige estas traducciones en [Crowdin](https://crowdin.com/project/kingdom-hall-attendant/). ¡Gracias!
